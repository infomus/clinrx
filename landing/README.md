# ClinRx

Simple launch-page repo for ClinRx.

## What’s here

- Single-page static landing page
- Brand-matched favicon + Apple touch icon
- Open Graph social sharing image
- GitHub Pages deploy workflow

## Local preview

```bash
cd clinrx
python3 -m http.server 8000
```

Then open <http://127.0.0.1:8000>.

## Deploy

This repo includes a GitHub Actions workflow that deploys the site to GitHub Pages on pushes to `main`.
